# Weekly Challenges - 20 Habit Challenges

This file contains 20 pre-defined weekly challenges based on habits from top CEOs, founders, biohackers, and proven life improvement practices. Each challenge runs for 7 days (Monday-Sunday) and participants must complete the minimum number of days specified.

## Challenge Format

```typescript
{
  id: number,              // Challenge number (1-20)
  name: string,            // Challenge name
  description: string,     // Full description
  dailyRequirement: string, // What to do each day
  minimalDose: string,     // Minimal acceptable completion
  daysRequired: number,    // Minimum days to complete (5-7)
  category: string,        // CEO | Biohacking | Life Improvement | Productivity | Health
  source: string,          // Attribution (CEO name, researcher, etc.)
  link: string            // External resource link
}
```

## The 20 Challenges

### 1. Deep Work Sessions
**Description:** Engage in distraction-free, cognitively demanding work for extended periods. Deep work is the ability to focus without distraction on a cognitively demanding task, producing high-quality output in less time.

**Daily Requirement:** 90 minutes of uninterrupted deep work
**Minimal Dose:** 60 minutes
**Days Required:** 5
**Category:** Productivity
**Source:** Cal Newport (Author, Computer Science Professor)
**Link:** https://www.calnewport.com/books/deep-work/

---

### 2. Morning Pages Journaling
**Description:** Write three pages of longhand, stream-of-consciousness writing first thing in the morning. This practice clears mental clutter and enhances creativity.

**Daily Requirement:** 3 pages (750 words) of morning journaling
**Minimal Dose:** 2 pages (500 words)
**Days Required:** 7
**Category:** Life Improvement
**Source:** Julia Cameron (Author, "The Artist's Way")
**Link:** https://juliacameronlive.com/basic-tools/morning-pages/

---

### 3. Cold Exposure Therapy
**Description:** Deliberate cold exposure through cold showers or ice baths. Increases metabolic rate, improves mood, and builds mental resilience.

**Daily Requirement:** 3 minutes of cold shower or 2 minutes ice bath

**Days Required:** 5
**Category:** Biohacking
**Source:** Wim Hof (Extreme Athlete, "The Iceman")
**Link:** https://www.wimhofmethod.com/cold-therapy

## Challenge Rotation Order

Challenges rotate in sequence from 1 → 2 → 3 → ... → 20 → 1, repeating indefinitely. Each challenge runs for one week (Sunday 3 PM deployment through following Sunday 9 AM evaluation).

## Notes for Implementation

- Users can only join on Sunday when the challenge is posted
- Maximum 1 proof per day allowed
- Minimal dose proofs count toward completion
- Completion requires meeting the "Days Required" threshold
- Successful completion earns €1 credit to challenge reward pool
- Challenges are designed to be accessible yet meaningful
